Справка по файловому менеджеру.

* путь в консольных командах нужно брать в кавычки, если он содержит пробелы
* прервать постраничный показ можно кнопкой Esc
* выход из файлового менеджера по кнопке F10
* следующая страница в постраничном режиме Enter
* ошибки пишутся в файл errors/log_exception.txt
* текущий выбранный путь записан в path.json

Доступные команды:
copy <файлОткуда> <файлКуда> -- копирует файл из <файлОткуда> в <файлКуда>
copy <папкаОткуда> <папкаКуда> -- копирует содержимое папки из <папкаОткуда> в <папкаКуда>
delete <файл/папка> -- удаляет папку с содержимым
info <файл/папка> -- показывает информацию о файле/папке
cd <папка> -- переходит в указанную папку и показывает ее содержимое в постраничном режиме. Если не указывать папку, заново покажет содержимое текущей папки.
tree <папка> -- показывает дерево
